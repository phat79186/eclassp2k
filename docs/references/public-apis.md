# Public APIs Reference

> Source: https://github.com/public-apis/public-apis
> Curated list of free public APIs — no auth or apiKey required unless noted.

## APIs hữu ích cho E-Class P2K

### 📅 Calendar / Lịch
| API | Mô tả | Auth | HTTPS | CORS |
|---|---|---|---|---|
| [Nager.Date](https://date.nager.at) | Ngày lễ công cộng 90+ quốc gia | No | Yes | No |
| [caldays](https://caldays.com/api) | Ngày lễ 195+ quốc gia | No | Yes | Yes |
| [Google Calendar](https://developers.google.com/google-apps/calendar/) | Tạo/hiển thị sự kiện Google Calendar | OAuth | Yes | - |

### 🔐 Authentication

| [GetOTP](https://otp.dev/en/docs/) | OTP flow nhanh | apiKey | Yes | No |
